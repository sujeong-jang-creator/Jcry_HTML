- 온프레미스(On-premise)란?
: 기업의 서버를 클라우드 같은 원격 환경에서 운영하는 방식이 아닌, 자체적으로 보유한 전산실 서버에 직접 설치해 운영하는 방식. 클라우드 컴퓨팅 기술이 나오기 전까지 기업 인프라 구축의 일반적인 방식.

- 온프라미스 용어 = AWS 용어
  방화벽 = 보안그룹
  ACL = NACL
  관리자 권한 = IAM
  L4, 로드 밸런서 = ELB(이라스틱 로드 밸런서), 탄력적인 로드 밸런서
  네트워크 = VPC
  서버 = EC2 (일라스틱 컴퓨트 클라우드, 탄력적인 서버)
  NAS = EFS (탄력적인 파일 시스템)
  디스크 = EBS (탄력적인 블록 스토어 저장장치)
  DB = RDS (관계형 데이터 베이스)
